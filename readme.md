### Heap layout

```
&heap
|                                                                                       &dhere
|                                                                                       |
v                                                                                       v
code...;|code....;|code...;| ...FREE...  |quotation...;|len.string...0|array...|var|var|dhere|
^        ^         ^        ^           ^ ^             ^              ^        ^   ^   |     ^
|        |         |        |           | |             |              |        |   |   |     |
|        |         |        \-> here    \-+-------------+--------------+--------+---+---/     |
|        |         \-> entry            | |             |              |        |   |         |
|        \-> entry                      | \-> quotation |              \-> var  |   \-> var   |
\-> entry                               |               \-> string              \-> var       |
                                        \-> dhere                                             |
                                                                                End of Heap <-/
```
